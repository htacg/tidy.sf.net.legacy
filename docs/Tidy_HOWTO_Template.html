Abstract

Provide an abstract giving context and purpose for the document...

This HOWTO provides a simple template for creating Tidy HOWTOs, using DocBook XML. Even if you don't author your HOWTO in XML, please follow something close to this layout.

This is the HTML output from the XML file. You can get the XML version of this file from http://prdownloads.sourceforge.net/tidy/Tidy_HOWTO_Template.xml.

If you think of something that should be added to this template, please email <teague@mailandnews.com>.

URLs

To specify a URL that will be hot-linked in HTML output, use the <ulink> element. In general, it is good practice to display the entire URL for readers.
